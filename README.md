# BodyBalance

Een Weight Watchers-stijl PWA die calorieën omzet in een simpel puntensysteem. Gebouwd voor iPhone (iOS) met React, TypeScript en Tailwind CSS.

## Features

- Persoonlijk dagelijks puntenbudget op basis van profiel
- Voedsel zoeken via Open Food Facts API
- Barcode scanner
- Maaltijd logging (ontbijt, lunch, diner, snacks)
- Gewichtstracker met grafiek
- 350+ ZeroPoint voedingsmiddelen
- Offline ondersteuning (PWA)

## Tech Stack

React 19 · TypeScript · Vite · Tailwind CSS 4 · Zustand · Dexie.js · Recharts

## Ontwikkeling

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push naar `main` → GitHub Actions deployt automatisch naar GitHub Pages.
